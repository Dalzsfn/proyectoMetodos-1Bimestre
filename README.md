#  Proyecto Ahorro Semanal

Bienvenido al repositorio central del **Proyecto Ahorro Semanal**, una iniciativa desarrollada para aplicar métodos numéricos en un contexto financiero real.  
Este repositorio integra el informe final, el simulador, el manual de usuario y la presentación oficial del proyecto.

---

#  Componentes del Proyecto

1. **Código Fuente**  
   El programa principal desarrollado en Python que implementa el cálculo de la tasa de interés usando el Método de la Secante, generando además la tabla y gráfica de evolución del ahorro.

2. **Presentación del Proyecto**  
   Exposición visual donde se explican objetivos, modelo matemático y resultados.

3. **Informe Final**  
   Documento detallado con el desarrollo matemático, deducción del modelo, implementación del método numérico y conclusiones.

---

#  Contenido del Repositorio

-  **Informe Final**  
  Explica el desarrollo matemático del modelo de ahorro, la ecuación no lineal y la aplicación del Método de la Secante.

-  **Presentación**  
  Resumen visual del propósito del proyecto y del simulador.

-  **Código Fuente del Simulador**

#  Objetivo del Proyecto

Aplicar el **Método Numérico de la Secante** para resolver la ecuación no lineal asociada al cálculo de la **tasa de interés por período** de un plan de ahorro con aportes periódicos, mostrando además la evolución del ahorro mediante una tabla y una gráfica.
